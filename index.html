Conceptos básicos y terminología

  • ‎main axis‎‎ – ‎ El main axis de un contenedor flexible es el main axis a lo largo del cual se colocan los elementos flexibles. Cuidado, no es necesariamente horizontal; depende de la propiedad (ver más abajo).‎ flex-direction
  • ‎main-start | main-end‎‎: los elementos flexibles se colocan dentro del contenedor comenzando desde el inicio principal y yendo al extremo principal.‎
  • ‎main size‎‎: el ancho o la altura de un elemento flexible, lo que sea que esté en la dimensión principal, es el main size del artículo. La propiedad de main size del elemento flexible es la propiedad 'width' o 'height', la que esté en la dimensión principal.‎
  • ‎cross main‎‎: el eje perpendicular al main axis se denomina eje transversal. Su dirección depende de la dirección del main axis.‎
  • ‎cross-start | cross-end‎‎: las líneas flex se llenan con elementos y se colocan en el contenedor comenzando en el lado de cross-start del contenedor flex y yendo hacia el lado cross-end.‎
  • ‎cross size‎‎: el ancho o alto de un elemento flexible, cualquiera que sea la dimensión de cruz, es el cross size del elemento. La propiedad de cross size es cualquiera de las opciones de 'anchura' o 'altura' que se encuentre en la cota transversal.‎
